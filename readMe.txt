Aplikasi penghitungan BMI sederhana menggunakan html dan css
pastikan anda sudah download Xampp
--------------STEP UNTUK DOWNLOAD--------------
1. Download file ini simple-BMI-kalkulator
2. ekstrak filenya kemudian masukkan kedalam folder htdocs didalam folder xampp kalian
3. buka xampp - Start Apache dan Mysql
4. setelah apache dan mysql start, buka browser dan ketikkan localhost/simple-BMI-kalkulator
5. aplikasi sudah bisa dijalankan.
